# 校园二手平台 — 功能总结清单

> 项目名称：校园二手（PBL）  
> 技术栈：Vue 3 + Vite + Ant Design Vue + Pinia | Express + MySQL + Socket.IO  
> 文档版本：2026-06-18

---

## 一、项目概述

| 项目 | 说明 |
|------|------|
| 产品定位 | 面向校园师生的二手交易平台，支持浏览、发布、购物车、下单、即时聊天与 AI 购物助手 |
| 品牌调性 | 年轻 · 可信 · 就近（Warm Campus Teal 主题色 `#0f857b`） |
| 前端地址 | `http://localhost:5173` |
| 后端地址 | `http://localhost:3000` |
| 数据库 | MySQL `2_shou` |

---

## 二、用户角色

| 角色 | 说明 |
|------|------|
| 访客 | 未登录，仅可访问 `/login`、`/register` |
| 普通用户 | 浏览、发布、交易、聊天（可同时买卖） |
| 管理员 | `role=1`，管理商品、用户、平台通知 |
| 买家/卖家 | 同一账号按业务场景区分，无独立角色切换 |

**默认管理员：** 用户名 `admin`，密码 `admin123`（首次启动自动创建或提升权限）

| 10 | 卖家发货 | 订单详情 / `/orders/pending-ship` | `POST /api/orders/:id/ship` | ✅ |

**订单状态流转：**

```
待付款(0) → 待发货(1) → 待收货(2) → 已完成(3)
    ↓
 已取消(4)（仅待付款可取消）
```

**付款后：** 关联商品状态变为「已售」(2)。

## 五、数据库表

| 表名 | 用途 |
|------|------|
| `users` | 用户（username, password, avatar, role, status, create_time） |
| `goods` | 商品（content, price, images, status, user_id） |
| `messages` | 聊天消息 |
| `cart_items` | 购物车 |
| `orders` | 订单主表 |
| `order_items` | 订单明细 |
| `addresses` | 收货地址（表已建，前端暂未独立管理） |
| `notifications` | 平台通知（title, content, type, is_active） |

启动后端时自动执行 `migrate.sql` 建表。

## 六、已知限制与待完善项

| 类别 | 说明 |
|------|------|
| 安全 | 密码明文、无 JWT；管理员鉴权依赖 admin_id 参数 |
| 支付 | 模拟付款，无真实支付网关 |
| 地址 | 结算页手动填写，未接入地址簿 |
| 搜索 | 仅按商品描述模糊匹配，无分类/排序 |
| 通知 | 无订单状态推送；平台通知仅首页展示 |
| 认证 | 无手机号/校园身份认证 |
