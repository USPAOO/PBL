校园二手交易平台 User Story 清单（带优先级）
P0（必做）- MVP 核心功能
US-001：校园身份认证
用户角色：平台运营方用户故事：As a 平台运营方I want 用户通过学工号 / 教职工号完成校园身份认证So that 确保平台用户的真实性和安全性
验收标准：
 用户可输入学工号 / 教职工号进行身份验证
 系统对接校园身份认证接口或支持人工审核
 认证通过后方可使用交易功能
 认证信息仅用于身份核验，保护用户隐私
US-002：用户注册 / 登录
用户角色：学生 / 教职工（所有用户）用户故事：As a 学生 / 教职工I want 通过手机号 + 验证码完成注册 / 登录，并绑定校园身份So that 安全便捷地使用平台功能，确保账号与校园身份关联
验收标准：
 支持手机号 + 验证码一键登录 / 注册
 登录后引导完成校园身份绑定，未绑定则限制交易功能
 支持记住登录状态（可选），退出登录功能正常
 登录异常（如异地登录）触发安全提醒
US-003：发布二手商品
用户角色：学生卖家 / 教职工卖家用户故事：As a 学生卖家 / 教职工卖家I want 快速发布我的闲置物品信息（含图片、描述、价格等）So that 让潜在买家看到并购买我的商品
验收标准：
 支持上传 1-9 张商品图片，单张不超过 5MB
 必填字段：标题、分类、价格、商品描述
 选填字段：原价、新旧程度、取货方式
 发布成功后商品进入审核状态，审核通过后展示
 支持保存草稿功能
US-004：商品编辑 / 下架
用户角色：学生卖家 / 教职工卖家用户故事：As a 学生卖家 / 教职工卖家I want 编辑已发布商品的信息或下架商品So that 及时更新商品状态，避免无效咨询
验收标准：
 仅商品发布者可编辑 / 下架自己的商品
 编辑功能支持修改除 “交易状态” 外的所有商品信息
 下架商品自动从列表隐藏，卖家可在 “我的商品” 中查看
 编辑后商品需重新进入审核流程
US-005：担保交易购买
用户角色：学生买家 / 教职工买家用户故事：As a 学生买家 / 教职工买家I want 使用担保交易购买商品（资金冻结→确认收货→资金释放）So that 我的资金安全有保障，避免被骗
验收标准：
 买家下单后，资金进入平台冻结账户
 卖家收到订单通知，准备发货 / 约定取货
 买家确认收货后，资金自动释放给卖家
 支持 7 天未确认自动确认收货机制
 交易过程中可申请退款 / 退货，进入仲裁流程
US-006：订单管理
用户角色：买家 / 卖家用户故事：As a 买家 / 卖家I want 查看订单状态、订单详情，必要时取消订单So that 掌握交易进度，自主管理订单
验收标准：
 买家端：查看待付款、待收货、已完成、已取消订单
 卖家端：查看待发货、待确认、已完成、已取消订单
 订单详情展示商品、价格、交易双方、进度等完整信息
 未付款 / 未发货状态下支持取消订单，取消后资金原路退回
US-007：买卖双方站内沟通
用户角色：买家 / 卖家用户故事：As a 买卖双方I want 通过平台内置消息系统沟通（文字 + 图片）So that 协商交易细节，保留沟通记录用于纠纷处理
验收标准：
 买家可对感兴趣的商品发起咨询
 支持发送文字消息和图片
 消息实时推送，未读消息有提醒
 聊天记录与订单关联，便于纠纷处理
 禁止发送联系方式（手机号、微信等），引导站内交易
US-008：商品搜索与筛选
用户角色：学生买家 / 教职工买家用户故事：As a 学生买家 / 教职工买家I want 通过关键词和条件（分类、价格、新旧）筛选找到需要的商品So that 快速定位目标商品，提高购物效率
验收标准：
 支持关键词搜索商品标题和描述
 支持按分类、价格区间、新旧程度筛选
 支持按发布时间、价格排序
 搜索结果支持分页展示，每页 20 条
 无结果时显示推荐商品或搜索建议
US-009：商品详情查看
用户角色：学生买家 / 教职工买家用户故事：As a 学生买家 / 教职工买家I want 查看商品完整信息（图片、描述、卖家信息、相似推荐）So that 全面了解商品，判断是否购买
验收标准：
 商品详情页展示所有发布信息（图片、标题、价格、描述、取货方式等）
 展示卖家昵称、信用等级、已成交数等基础信息
 底部推荐 3-5 个相似分类 / 价格的商品
 支持从详情页发起咨询、收藏、购买操作
US-010：商品举报
用户角色：所有用户用户故事：As a 平台用户I want 举报违规商品或违规行为So that 维护平台秩序，避免不良信息干扰
验收标准：
 商品详情页 / 聊天页提供举报入口
 举报需选择类型（虚假信息、禁售商品、骚扰等），可选填描述
 举报后信息提交至运营后台，用户可查看举报进度（仅自己可见）
 举报成功后平台隐藏违规内容（审核确认后）
US-011：后台商品审核
用户角色：平台运营方用户故事：As a 平台运营方I want 审核用户发布的商品、处理用户举报So that 确保平台商品合规，无违规信息
验收标准：
 后台展示待审核商品列表，支持批量 / 单个审核
 审核选项：通过、驳回（需填写驳回理由）
 展示举报列表，支持查看举报详情、处理举报（警告 / 下架 / 封号）
 审核操作留有日志，便于追溯
P1（重要）- 体验优化功能
US-012：交易评价
用户角色：买家 / 卖家用户故事：As a 交易双方I want 在交易完成后互相评价（星级 + 文字）So that 建立信用体系，帮助其他用户判断卖家 / 买家可信度
验收标准：
 交易完成后 7 天内可进行评价
 评价维度：描述相符、交易速度、服务态度（星级评分）
 支持文字评价和星级评分（1-5 星）
 评价内容公开显示，不可修改
 累计评价形成用户信用等级
US-013：商品收藏与降价提醒
用户角色：学生买家 / 教职工买家用户故事：As a 学生买家 / 教职工买家I want 收藏感兴趣的商品，并接收商品降价提醒So that 在价格合适时及时购买
验收标准：
 用户可收藏 / 取消收藏商品，收藏列表支持分类查看
 收藏商品降价时，向用户推送站内通知
 商品下架 / 售出时自动从收藏列表移除，并推送通知
 收藏列表支持搜索、筛选功能
US-014：商品价格参考
用户角色：学生卖家 / 教职工卖家用户故事：As a 学生卖家 / 教职工卖家I want 基于平台同类商品获得价格参考So that 合理定价，提高商品成交率
验收标准：
 发布商品时，系统根据分类、新旧程度、品牌等推荐价格区间
 展示近 30 天同类商品的成交均价，供卖家参考
 价格参考数据仅作提示，不强制卖家定价
US-015：议价功能
用户角色：买家 / 卖家用户故事：As a 买家I want 向卖家发起议价请求As a 卖家I want 同意 / 拒绝买家的议价请求So that 灵活协商商品价格，提升交易意愿
验收标准：
 买家可在商品详情页 / 聊天页发起议价（输入期望价格）
 卖家收到议价通知，可同意（生成新价格）或拒绝（可选填理由）
 议价成功后，买家可按新价格下单
 单商品最多支持 3 次议价，避免恶意砍价
US-016：个性化商品推荐
用户角色：所有用户用户故事：As a 平台用户I want 基于我的浏览历史和偏好推荐商品So that 发现更多符合需求的商品，提升使用体验
验收标准：
 首页展示 “为你推荐” 板块，基于浏览 / 收藏 / 交易记录推荐
 推荐商品与用户偏好匹配度≥60%（后期数据验证）
 支持 “不感兴趣” 操作，减少同类商品推荐
US-017：热门商品展示
用户角色：所有用户用户故事：As a 平台用户I want 查看平台热门商品、新品上架商品So that 了解平台交易趋势，发现高性价比商品
验收标准：
 首页展示 “热门商品”（按浏览 / 咨询量排序）、“新品上架”（按发布时间排序）板块
 板块支持下拉刷新，数据实时更新
 热门商品按分类展示，支持点击进入分类列表
US-018：站内消息通知
用户角色：所有用户用户故事：As a 平台用户I want 接收订单状态变更、新消息、系统通知等提醒So that 及时掌握交易进度和平台动态
验收标准：
 订单付款、发货、确认收货等状态变更触发站内通知
 新消息、举报处理结果、系统公告等实时推送
 未读通知有红点 / 数字提醒，支持一键已读
 通知列表支持筛选（订单 / 消息 / 系统）
P2（可选）- 增值功能
US-019：关注卖家
用户角色：买家用户故事：As a 学生买家 / 教职工买家I want 关注喜欢的卖家So that 及时获取卖家的上新通知，方便复购
验收标准：
 卖家主页支持 “关注 / 取消关注” 操作
 关注后卖家上新商品时，向买家推送通知
 买家可在 “我的关注” 中查看已关注卖家及上新动态
US-020：求购发布
用户角色：学生买家 / 教职工买家用户故事：As a 学生买家 / 教职工买家I want 发布求购需求（如 “求购二手自行车”）So that 让卖家主动对接，快速找到需要的商品
验收标准：
 支持发布求购信息（标题、分类、期望价格、描述）
 求购信息展示在 “求购专区”，支持卖家报名对接
 买家可查看报名卖家列表，主动沟通
 求购信息支持下架 / 完成（找到商品后）
US-021：商品置顶推广
用户角色：学生卖家 / 教职工卖家用户故事：As a 学生卖家 / 教职工卖家I want 付费将商品置顶展示So that 提高商品曝光率，加快成交速度
验收标准：
 商品发布 / 编辑页支持 “置顶推广” 选项，展示置顶价格和时长
 付费后商品在分类列表 / 搜索结果中优先展示，标注 “置顶” 标签
 置顶时长结束后，商品自动恢复普通排序
 卖家可查看置顶效果（曝光量、咨询量提升数据）
US-022：验货服务
用户角色：买家 / 卖家用户故事：As a 买家 / 卖家I want 申请平台提供商品验货担保服务So that 确认商品质量，降低交易纠纷风险
验收标准：
 支持申请平台验货（仅限电子产品、贵重物品等），验货需支付服务费
 平台验货后出具验货报告（含功能、成色、瑕疵等），双方可见
 验货通过后再进行担保交易，纠纷时以验货报告为仲裁依据
 验货服务仅支持校内指定地点完成
US-023：拍卖模式
用户角色：卖家 / 买家用户故事：As a 卖家I want 以拍卖模式发布商品，允许买家竞价As a 买家I want 参与商品竞价，以合理价格拍得商品So that 实现商品价值最大化（卖家）/ 以心仪价格购买（买家）
验收标准：
 卖家发布商品时可选择 “拍卖模式”，设置起拍价、加价幅度、拍卖时长
 买家可在拍卖时长内出价，出价后实时展示当前最高价
 拍卖结束后，最高价买家生成订单，需在 24 小时内付款
 拍卖过程中支持 “一口价” 直接成交（卖家可选开启）
US-024：以物易物
用户角色：学生 / 教职工用户故事：As a 学生 / 教职工I want 发布物品交换需求，与其他用户交换闲置So that 无需现金交易，实现资源互补
验收标准：
 支持发布 “以物易物” 类型商品，标注期望交换的物品
 交换需求展示在 “以物易物” 专区，支持筛选 / 搜索
 双方确认交换后，生成交换订单，支持担保交换（先寄 / 交物品至平台，确认后完成交换）
 交换完成后支持互评，计入信用体系
