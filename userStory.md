校园二手交易平台 - 用户故事文档
文档版本：V1.0生成日期：2026-04-07依据：校园二手交易平台需求分析文档 V1.0优先级说明：P0 = 必做（MVP）、P1 = 重要、P2 = 可选
一、P0 核心功能用户故事（MVP 必做）
US-P0-001 校园身份认证
As a 平台运营方I want 用户通过学工号 / 教职工号完成校园身份认证So that 确保平台用户真实可信，防范校外无关人员与欺诈风险
验收标准
用户可输入学工号 / 教职工号进行身份验证
支持对接校园统一认证接口或人工审核
未认证用户仅可浏览，不可发布 / 交易 / 沟通
认证信息加密存储，仅用于核验，不对外泄露
US-P0-002 用户注册与登录
As a 校园用户（学生 / 教职工）I want 使用手机号 + 验证码快速注册登录，并绑定校园身份So that 安全进入平台，使用完整交易功能
验收标准
支持手机号获取验证码登录 / 注册
登录后强制引导完成校园身份认证
支持退出登录、修改密码
登录状态持久化，短时间内免重复登录
US-P0-003 个人资料管理
As a 平台用户I want 管理我的头像、昵称、联系方式、取货地址So that 交易时信息准确，方便买卖双方对接
验收标准
可修改头像、昵称、展示姓名
可添加 / 编辑 / 删除取货 / 收货地址
隐私信息仅自己可见，必要信息脱敏展示
US-P0-004 商品发布
As a 学生 / 教职工卖家I want 快速发布闲置商品，上传图片、填写描述与价格So that 买家能看到我的商品并咨询购买
验收标准
支持上传 1–9 张商品图片，单张≤5MB
必填：标题、分类、价格、商品描述
选填：原价、新旧程度、取货方式
支持保存草稿，发布后进入审核
审核通过后在列表正常展示
US-P0-005 商品编辑与下架
As a 商品卖家I want 修改商品信息或手动下架商品So that 及时更新状态，避免已售商品仍被下单
验收标准
可编辑标题、价格、描述、图片等信息
编辑后重新进入审核
可一键下架 / 上架，下架后不展示
已产生订单的商品不允许删除，仅可下架
US-P0-006 担保交易购买
As a 学生买家I want 使用担保交易下单支付So that 资金安全，确认收货后卖家才收款
验收标准
买家下单付款 → 资金冻结
卖家发货 / 约定面交 → 订单更新待收货
买家确认收货 → 资金转给卖家
7 天未确认自动确认收货
支持申请退款 / 退货，进入平台仲裁
US-P0-007 订单管理
As a 买家 / 卖家I want 查看所有订单状态与详情So that 清晰掌握交易进度，处理异常
验收标准
订单状态：待付款、待发货 / 待取货、待收货、已完成、已取消
买家可取消未付款订单
卖家可查看待处理订单与收款进度
订单展示商品、金额、时间、双方信息（脱敏）
US-P0-008 站内即时通讯
As a 买家 / 卖家I want 在平台内直接沟通交易细节So that 不依赖外部软件，留痕便于纠纷处理
验收标准
可从商品页发起聊天
支持文字、图片发送
未读消息提醒，实时推送
聊天记录与订单绑定
自动屏蔽手机号、微信、QQ 等外部联系方式
US-P0-009 商品搜索与筛选
As a 学生买家I want 快速搜索并筛选商品So that 高效找到想要的物品
验收标准
支持关键词搜索标题 / 描述
支持按分类、价格区间、新旧程度筛选
支持按最新发布、价格升降序排序
分页展示，每页 20 条
无结果时给出推荐与提示
US-P0-010 商品列表与详情
As a 平台用户I want 浏览商品列表并查看完整详情So that 了解商品与卖家信息，决定是否购买
验收标准
列表展示图片、标题、价格、发布时间
详情展示所有商品信息、卖家信息、交易方式
提供相似商品推荐
图片支持预览、缩放
US-P0-011 用户举报功能
As a 平台用户I want 举报违规商品 / 不良行为So that 维护平台安全干净的环境
验收标准
支持举报商品、举报用户
需选择原因并可上传凭证
提交后显示处理中
处理完成推送结果通知
US-P0-012 后台审核与管理
As a 平台运营方I want 在后台审核商品、处理举报、管理用户So that 控制内容风险，处理平台纠纷
验收标准
后台可查看待审核商品、待处理举报
可通过 / 驳回，驳回需填写原因
可查看订单、用户数据
支持按条件筛选与导出
二、P1 重要功能用户故事（体验优化）
US-P1-001 交易评价与信用体系
As a 交易完成的买卖双方I want 对交易进行互评So that 建立信用，帮助他人判断可信度
验收标准
交易完成 7 天内可评价
星级：描述相符、交易速度、服务态度
支持文字评价
评价公开，形成信用等级
恶意评价可申诉
US-P1-002 商品收藏
As a 买家I want 收藏感兴趣的商品So that 后续方便找到，不错过合适商品
验收标准
详情页一键收藏 / 取消收藏
个人中心展示收藏列表
商品下架自动移除并提示
US-P1-003 降价订阅提醒
As a 买家I want 收藏商品降价时收到通知So that 价格合适时及时购买
验收标准
商品降价触发站内通知
可开关通知权限
展示降价幅度与原 / 现价
US-P1-004 价格参考
As a 卖家I want 发布商品时看到同类价格参考So that 定价更合理，更快卖出
验收标准
按分类 + 新旧程度给出参考区间
基于平台真实成交数据
仅作参考，不强制定价
US-P1-005 议价功能
As a 买家I want 向卖家发起议价So that 不用反复聊天，高效达成价格一致
验收标准
买家输入意向价发起议价
卖家可同意 / 拒绝 /counter 还价
同意后自动生成对应价格订单
US-P1-006 个性化推荐
As a 买家I want 平台推荐我可能感兴趣的商品So that 发现好物，提升交易效率
验收标准
基于浏览、收藏、搜索记录推荐
首页展示热门、新品、为你推荐
US-P1-007 站内消息通知
As a 平台用户I want 收到订单、消息、系统通知So that 不错过关键交易信息
验收标准
通知：新消息、订单状态、审核结果、举报结果
未读红点提示
可设置通知开关
三、P2 可选功能用户故事（增值扩展）
US-P2-001 关注卖家
As a 买家I want 关注靠谱卖家So that 及时看到他的新上架商品
US-P2-002 校园圈子 / 社区
As a 学生用户I want 进入兴趣圈子So that 同好交流，更精准交易（考研、骑行、数码等）
US-P2-003 商品付费置顶
As a 卖家I want 付费让商品排在前面So that 提高曝光，更快成交
US-P2-004 平台验货服务
As a 买卖双方I want 平台提供验货担保So that 降低数码等高价值商品交易风险
US-P2-005 求购发布
As a 买家I want 发布求购信息So that 卖家主动联系我
US-P2-006 拍卖模式
As a 卖家I want 用拍卖方式出售稀缺物品So that 卖出更合理价格
US-P2-007 以物易物
As a 校园用户I want 用闲置交换他人物品So that 不花钱实现资源互换
四、文档使用说明
P0 为最小可用版本（MVP），必须全部开发完成方可上线
P1 为体验优化项，建议 MVP 上线后 1–2 个迭代完成
P2 为增值功能，根据资源、运营数据与用户需求酌情开发
每条用户故事可直接拆解为开发任务、测试用例与验收标准
