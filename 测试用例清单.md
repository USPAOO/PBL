> 测试环境：Chrome / Edge 最新版；后端 `localhost:3000`；前端 `localhost:5173`

## 十一、端到端核心流程（冒烟测试）

按顺序执行，验证主链路贯通：

```
冒烟流程 A — 买家完整购物流程
─────────────────────────────
1. [AUTH-01] 买家登录
2. [HOME-05] 浏览并进入商品详情
3. [GD-03]   加入购物车
4. [CART-04] 购物车结算
5. [ORD-01]  提交订单
6. [ORD-05]  付款
7. [ORD-09]  确认收货（需卖家先 ORD-08 发货）

冒烟流程 B — 卖家发布与履约
─────────────────────────────
1. [AUTH-01] 卖家登录
2. [PUB-01]  发布商品
3. （买家下单付款后）
4. [ORD-08]  卖家发货

冒烟流程 C — 沟通与助手
─────────────────────────────
1. [GD-06]   联系卖家聊天
2. [MSG-01]  互发消息
3. [AGT-01]  购物助手推荐
```
